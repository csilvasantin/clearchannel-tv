admira.app/tutorial

Configura Target Model y convierte paso de personas en inventario vendible

Este tutorial guia la operacion diaria: definir personas por hora, ajustar segmentos, calibrar surfaces y publicar reglas de reproduccion para campañas segmentadas.

Flujo operativo

Seis pasos para publicar un target

Paso 01

Selecciona circuito y Xpacio

Empieza por el circuito comercial y el punto fisico. Target Model carga el baseline de trafico, superficies disponibles y ultimo estado sincronizado con Admira.

  • Comprueba exterior/interior.
  • Revisa rango de fechas.
  • Elige modo simulado u observado.

Metricas clave

Que mirar antes de vender una campaña

01

Personas/hora

Es el volumen base del gemelo digital. Cada hora puede tener fuente manual, estimada u observada.

02

Audiencia util

Porcentaje del trafico que coincide con placement, edad, genero, franja y metadata seleccionada.

03

Impresiones

Personas alcanzables multiplicadas por visibilidad, dwell time y frecuencia maxima por surface.

04

Revenue

Valor comercial calculado con CPM, dias de campaña y cobertura del target activo.

Privacidad por diseño

Trabaja siempre con datos agregados

Target Model debe guardar conteos, porcentajes y estimaciones. No debe crear fichas personales ni usar datos sensibles para identificar individuos. Si se conectan camaras, sensores o señales de dispositivo, la salida operativa debe ser agregada y auditable.
